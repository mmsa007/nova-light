🤖 OpenClaw

你的 AI 助手 - 强大的自动化框架，支持多平台接入和 Skills 扩展

什么是 OpenClaw？

OpenClaw 是一个开源的 AI 助手框架，可以帮你完成各种任务：搜索信息、生成图片、操作浏览器、管理文件等。它就像一个超级助手，可以接入微信、Telegram、Discord 等平台，还能通过 Skills 扩展各种能力。

✨ 核心功能

🔍 智能搜索

支持多种搜索引擎

🖼️ AI 生图

集成 DALL-E、豆包等

🌐 浏览器自动化

自动操作网页

💬 多平台接入

微信、Telegram、Discord

⚡ 定时任务

cron 定时执行

🔌 Skills 扩展

50+ Skills

📥 安装教程

macOS 安装：

brew install openclaw/tap/openclaw

启动：

openclaw start

更多请访问：openclaw.ai
